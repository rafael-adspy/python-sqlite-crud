# Sistema de Estoque

Sistema de controle de estoque desenvolvido em Python utilizando SQLite para armazenamento dos dados e Colorama para estilização da interface no terminal.

## Funcionalidades

* Cadastrar produtos
* Buscar produtos
* Atualizar produtos
* Excluir produtos
* Listar todos os produtos cadastrados
* Armazenamento persistente com SQLite

## Tecnologias Utilizadas

* Python 3
* SQLite3
* Colorama

## Estrutura do Projeto

```text
.
├── main.py
├── db.py
├── menu.py
├── cadastro.db
└── README.md
```

## Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/crud-estoque-python.git
```

Acesse a pasta do projeto:

```bash
cd crud-estoque-python
```

Instale as dependências:

```bash
pip install colorama
```

Execute o programa:

```bash
python main.py
```

## Demonstração

O sistema permite gerenciar produtos através de um menu interativo no terminal:

* Cadastro de produtos
* Consulta de produtos
* Atualização de informações
* Exclusão de produtos
* Listagem completa do estoque

## Objetivo

Este projeto foi desenvolvido com fins de estudo para praticar:
